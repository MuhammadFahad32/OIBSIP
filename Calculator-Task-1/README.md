# 🧮 Modern Calculator

A modern, responsive, and user-friendly calculator built using **HTML5, CSS3, and Vanilla JavaScript** as part of the **Oasis Infobyte Internship Program (OIBSIP)**.

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- 📊 Percentage Calculation
- 🔢 Decimal Number Support
- 🗑 Clear (AC) Button
- ⌫ Delete Last Character
- ⌨ Keyboard Support
- ⚠ Error Handling
- 📱 Fully Responsive Design
- 🎨 Modern Glassmorphism UI

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## 📂 Project Structure

```
Task-1-Calculator/
│── index.html
│── style.css
│── script.js
│── README.md
```

## 💡 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Run `index.html` in your browser.

No additional installation is required.
